# CountryPeek

CountryPeek is a React + Vite app for searching countries and viewing country information.

## Available Scripts

- `npm run dev` - start the development server
- `npm run build` - build for production
- `npm run preview` - preview the production build locally
- `npm run lint` - run ESLint

## Part 1 Scope

This part includes the initial app shell:

- React Router setup with routes for Home, Country placeholder, Favourites placeholder, and 404
- Shared `Header` navigation using `Link`
- `Home` page with a controlled `SearchBar` input shell
- `NotFound` fallback page
- Base global styles in `src/styles/index.css` and `src/styles/App.css`
